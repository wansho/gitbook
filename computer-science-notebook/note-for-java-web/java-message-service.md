## 是什么

JMS 是 Java 消息服务的应用程序接口，和 JDBC 一样，只定义接口，不做具体实现。

JMS 是 Java 对于消息队列 MQ 的一种实现方式。

JMS 的实现：ActiveMQ、RabbitMQ、RocketMQ，其中 RocketMQ 并没有完全遵循 JMS 规范。

注意，Kafka 不是 JMS 的实现。
