## Python 运算符

| 运算符 | 解释                            | Demo            |
| ------ | ------------------------------- | --------------- |

## Python 与 C 的语法差异

### 数值型 + 1

Python 不能通过 `++a` 实现自增。

```python
# C
int a = 1;
queue[++a] # == queue[2]
prinf("%d", a) # a == 2


# Python
a = 1
queue[++a] # == queue[2]
print(a) # 1
```

## 其他问题

### 指针和引用的区别

Pointer variable vs Reference variable

* 引用更像是一个**贴纸**，贴纸上写上内存地址空间的别名（非空），然后贴在这块地址空间上，是一次性的（不能重新赋值）

* 指针首先是一个**指针变量**，这个变量存储着一个地址空间，这个变量可以为空，也可以重新赋值
