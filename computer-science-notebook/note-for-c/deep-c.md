## 数据类型

C 语言中的数据类型可以理解为**固定内存大小的别名**，是**创建变量的模子**。

C 语言是一种弱类型语言，其一块内存可以叫做不同的类型。

内存空间按照大小可以命名为：

| 字节数 | 命名  |
| ------ | ----- |
| 1      | char  |
| 2      | short |
| 4      | int   |
